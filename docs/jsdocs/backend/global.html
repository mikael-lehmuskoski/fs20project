(async) login(_, args, context)

login

Uses mongoose and bcrypt to find and compare arguments to user found in database. The request is responded with an object containing the user's complete profile and a JWT token upon successful comparison.

(async) signup(_, args, context)

signup

Uses mongoose and bcrypt to create a new user if the username passed as an argument is unique. The request is responded with an object containing the user's complete profile and a JWT token upon successful creation.

Arguments are validated with JOI before operations.

User's password is stored as a hash in the database.
